# Hardware - Enclosures & Mounting

This directory contains 3D printable designs for node enclosures and mounting solutions.

## Designs
- `handheld_unit/` - Portable unit with onboard LCD
- `room_mounted/` - Fixed wall-mount WiFi node
- `antenna_holder/` - Antenna positioning brackets

## Materials
- PLA or PETG (recommended for durability)
- Print settings:
  - Layer height: 0.2mm
  - Infill: 20%
  - Supports: Yes, tree supports recommended

## Assembly Instructions
Each design includes:
- Detailed images and measurements
- Component list (PCB, antenna, fasteners)
- Assembly guide

## Notes
- Antenna placement is critical for CSI performance
- Handheld units should have ~5cm clearance above ground for best results
